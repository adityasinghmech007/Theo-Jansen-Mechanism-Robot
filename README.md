# Theo Jansen Walking Robot

## 📌 Overview

This project was developed as our Final Year Engineering Project, where our team designed and built a Theo Jansen-inspired walking robot capable of differential drive motion control.

The robot was completely designed in SolidWorks, including mechanism modeling and motion analysis. After validating the mechanical movement through motion studies, the parts were laser cut and assembled into a fully functional walking robot.

The system uses stepper motors driven through DRV8825 motor drivers and is controlled using an Arduino-based embedded control system powered by LiPo batteries.

## 🤖 Motion Control

The robot uses differential drive control to steer and navigate by independently controlling the speed of the two stepper motors.

### Functionalities

* Forward and reverse movement
* Turning using differential motor speeds
* Coordinated walking mechanism control
* Stable multi-link leg motion
